# 📊 Super Store Sales & Operations Analytics
### End-to-End BI Solution: From SQL Data Warehousing to Executive Storytelling.

## 📋 Descripción
Este proyecto resuelve la fragmentación de información en una empresa retail, transformando **+50k registros crudos** en una herramienta de decisión estratégica. Implementé un flujo de datos completo (ETL) asegurando integridad referencial y alto rendimiento en las consultas.

## 🛠 Tecnologías
* **Base de Datos:** MySQL (Modelado Relacional, Vistas Analíticas, Optimización de Queries).
* **BI Tool:** Power BI (DAX Avanzado, Modelado Dimensional, UI/UX Dark/Light Mode).
* **Lenguajes:** SQL, DAX.

## 🏗 Arquitectura del Proyecto
1. **Capa de Datos:** Limpieza y normalización de datos en MySQL.
2. **Modelo de Datos:** Esquema en Estrella con tablas de hechos (`fact_sales`) y dimensiones (`dim_locations`, `dim_customers`, `dim_products`).
3. **Capa Analítica:** Creación de vistas SQL para desacoplar la lógica de negocio del reporte.
4. **Visualización:** Dashboard de 3 niveles: Operativo, Clientes y Logística.

## 📈 Insights Clave & Valor de Negocio
* **Optimización de Margen:** Identificación de una brecha de rentabilidad en el segmento "Home Office".
* **Eficiencia Logística:** El impacto de envío se mantiene en **10.70%**, permitiendo ajustar estrategias de "Free Shipping" sin comprometer la utilidad.
* **Segmentación RFM:** Identificación de clientes de alto valor vs. clientes de alta frecuencia.
